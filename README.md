# ChessDex C++

A second attempt at coding a chess engine... this time with C++ instead of python.

My first attempt was [ChessDex](https://github.com/DexterCarpenter/ChessDex)

# Configure, Build, & Run

Run the following commands at the workspace root to configure the default preset, build the default preset, and run the default preset respectively.

```bash
cmake --preset defualt
cmake --build out/build/defualt
./out/build/defualt/ChessDex-cpp
```
